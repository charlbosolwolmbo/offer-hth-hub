# offer-hth-hub

一个用于归档和发布独立 HTML 页面的仓库。

## 项目简介

本项目旨在集中存放多个独立、静态的 HTML 页面。这些页面可能用于展示信息、提供工具、或作为临时网页使用。仓库本身不针对特定站点或服务，仅作为 HTML 文件的托管与版本管理空间。

通过 GitHub Pages 或类似服务，仓库中的页面可直接访问，便于分享与测试。

## 目录说明

```
/
├── index.html          # 仓库首页（可选）
├── pages/              # 存放独立 HTML 页面的目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，互不依赖。
- `assets/`：存放页面共享的样式、脚本或图片资源。

## 页面归档说明

- 所有页面均为纯静态 HTML，不依赖后端服务。
- 页面间的链接应使用相对路径，避免硬编码外部地址。
- 如需引用外部资源（如字体、库），建议使用 CDN 或将其下载至 `assets/` 目录。
- 页面文件命名建议使用英文小写，以短横线分隔（如 `my-page.html`）。

## 维护说明

- 欢迎提交 Pull Request 新增或修改页面，但请确保内容合规、无恶意代码。
- 提交前请检查 HTML 语法和资源路径是否正确。
- 不建议在页面中包含跟踪脚本、广告或敏感内容。
- 本仓库仅用于 HTML 页面的归档与展示，不承担其他用途责任。
